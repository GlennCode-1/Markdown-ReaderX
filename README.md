# 📚 Markdown-ReaderX

一个简洁Markdown阅读器，支持云存储、笔记和高亮功能。

## ✨ 核心特性

### 💾 智能云存储

- ✅ 使用IndexedDB本地存储
- ✅ 支持多文件管理
- ✅ 自动保存所有更改
- ✅ 完全离线可用

### 🖍️ 强大的标注功能

- ✅ 四色高亮标记系统
- ✅ 双击添加笔记
- ✅ 笔记与段落关联
- ✅ 一键导出所有笔记

## 🎯 使用方法

### 文件管理

- **上传**：点击左侧"➕ 上传文件"按钮
- **切换**：点击文件列表中的任意文件
- **删除**：悬停文件，点击垃圾桶图标

### 高亮标记

1. 点击工具栏"🖍️ 高亮"激活
2. 选中要标记的文字
3. 选择颜色（黄/绿/蓝/粉）
4. 自动保存

### 添加笔记

1. 双击任意段落
2. 输入笔记内容
3. 点击保存
4. 笔记显示在右侧面板

### 导出笔记

- 点击"📥 导出"按钮
- 自动生成Markdown格式
- 包含所有笔记和上下文

## 🌟 技术亮点

- **纯前端**：单文件HTML，无需服务器
- **零依赖**：仅使用CDN加载的marked.js和highlight.js
- **高性能**：IndexedDB提供快速存储
- **隐私保护**：所有数据仅保存在本地
- **响应式**：完美支持桌面和移动设备

## 💡 使用场景

- 📖 **学习笔记**：阅读技术文档，标记重点
- 📝 **论文研究**：高亮关键数据，添加理解
- 📚 **知识管理**：管理所有Markdown笔记
- ✍️ **写作辅助**：审阅和标注自己的文章

## 🔒 隐私与安全

- ✅ 所有文件仅保存在浏览器本地
- ✅ 不会上传任何数据到服务器
- ✅ 完全开源透明
- ✅ 可随时清除所有数据
